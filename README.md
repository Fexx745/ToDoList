To-Do List ที่พัฒนาด้วย React และ TypeScript โดยสามารถทำงานพื้นฐานได้ เช่น
- เพิ่ม / ลบ / แก้ไข Task
- ติ๊กเครื่องหมายว่าเสร็จแล้ว
- บันทึกรายการลง `localStorage` เพื่อให้ข้อมูลคงอยู่แม้รีเฟรชหน้าเว็บ

## วิธีเปิดชิ้นงาน 
1. ติดตั้ง dependencies: 
  - npm i
2. เปิดโปรเจกต์
  - npm run dev
